# MasterRepo Phase 4 EVA / Airlock / Tether Foundation Pack

This pack provides the first gameplay foundation for:
- EVA mode transitions
- airlock runtime logic
- tether state and supply logic
- exterior survival state
- zero-G movement shell
- interior/environment linkage
- exterior interaction targets

## Target loop
Walk inside ship -> interact with airlock -> cycle -> enter EVA -> tether active -> move in zero-G -> interact with exterior target -> return to airlock -> repressurize -> re-enter interior
