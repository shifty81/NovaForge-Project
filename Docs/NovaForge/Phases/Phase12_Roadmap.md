# Phase 12 Roadmap

## Core additions
1. Titan construction system
2. Endgame anomaly gate system
3. Seasonal state and reset model
4. Carryover reward and legacy bonus model
5. Large-scale resource sink and long-horizon objective layer

## Immediate intended result
- titan projects can track massive construction progress
- endgame gates can validate readiness and trigger transition state
- seasons can move from active -> ending -> reset
- selected rewards can carry into the next season
- full game loop becomes cyclical and replayable

## Best next integration after this pack
- feed mining, salvage, faction, and fleet outputs into titan requirements
- tie anomaly pressure and sector war into titan race pacing
- grant seasonal legacy rewards based on titan success
- use carryover bonuses to seed new seasonal starts
