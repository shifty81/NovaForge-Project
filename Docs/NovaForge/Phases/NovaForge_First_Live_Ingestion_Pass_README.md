# MasterRepo NovaForge First Live Ingestion Pass

This pack is the first controlled ingestion pass for NovaForge-style game data into Master Repo.

## Scope
This pass focuses on safe data ingestion only:
- items
- recipes
- missions
- factions
- loot tables

## Purpose
These files are ready to be dropped into the clean Master Repo data tree and used as:
- schema-aligned starter content
- test definitions for runtime loading
- reference outputs for future automated conversion from archived NovaForge content

## Rule
This is **data ingestion**, not direct legacy runtime import.
