# Phase 9 Roadmap

## Core additions
1. Station service system
2. Manufacturing queue system
3. Storage and cargo container system
4. Refining / processing hooks
5. Home-base interaction shell

## Immediate intended result
- stations can expose services
- player can deposit and withdraw inventory to storage
- manufacturing jobs can queue and complete
- station terminals can offer repair, resupply, trade, and crafting access
- return loop starts feeling meaningful

## Best next integration after this pack
- connect salvaged/mined items into storage
- refine ore into processed materials
- craft upgraded modules/tools from station manufacturing
- gate better services by faction standing and station tier
