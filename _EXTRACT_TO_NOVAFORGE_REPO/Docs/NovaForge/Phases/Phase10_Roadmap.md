# Phase 10 Roadmap

## Core additions
1. Ship progression and tiering
2. Fleet registry and assignment system
3. Patrol / escort / hauling / salvage task scaffolding
4. Meta-loop expansion state
5. Multi-ship ownership hooks

## Immediate intended result
- player can own more than one ship
- ships can track progression tier and role
- fleet can register members and assignments
- assignments can express patrol, hauling, salvage, escort, and mining roles
- project begins shifting from solo loop into operational expansion game

## Best next integration after this pack
- connect contracts to fleet assignments
- connect stations to ship storage and fleet docking
- apply upgrades and economy results to ship progression
- add AI crew/fleet automation later
