# MasterRepo DataRegistry Expansion Pack

This pack expands the Phase 2 runtime `DataRegistry` so it can load and report the first live NovaForge-ingested data categories:

- items
- recipes
- missions
- factions
- loot tables
- modules (existing support retained)

## Purpose
This is the bridge between:
- the clean Phase 2 runtime pack
- the NovaForge first live ingestion pass

## Recommended use
Merge these files into your current runtime project, then copy the ingested data folders into your live `/Data/Definitions/...` tree.
