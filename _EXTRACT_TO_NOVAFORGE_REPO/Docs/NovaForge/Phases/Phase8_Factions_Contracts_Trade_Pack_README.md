# MasterRepo Phase 8 Factions + Contracts + Trade Pack

This pack adds the first living-world simulation layer for Master Repo.

## Included
- faction state and reputation scaffolding
- contract / mission board scaffolding
- trade route and market shipment scaffolding
- lightweight world simulation controller
- starter faction, contract, trade, and world-sim definition files

## Goal
Turn the progression loop into a living-world loop:

Mine / Salvage / Craft -> Sell / Accept Contracts -> Gain Faction Standing -> Access Better Opportunities
