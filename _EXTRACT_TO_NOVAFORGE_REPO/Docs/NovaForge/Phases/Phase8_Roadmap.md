# Phase 8 Roadmap

## Core additions
1. Faction standing and influence
2. Contract board / job system
3. Trade route and shipment model
4. World simulation tick hooks
5. Reputation-gated opportunities

## Immediate intended result
- factions track standing
- contracts can be offered and accepted
- trade routes can define supply movement
- world sim can tick faction + market behavior
- player progression can start feeding into a living world

## Best next integration after this pack
- link completed salvage/mining jobs into faction reputation
- generate contract offers from market shortages
- use trade routes to modify economy demand/supply over time
- gate higher-tier contracts by standing and skill level
