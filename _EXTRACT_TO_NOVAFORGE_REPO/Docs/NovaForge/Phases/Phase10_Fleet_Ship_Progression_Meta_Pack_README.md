# MasterRepo Phase 10 Fleet + Ship Progression + Meta Loop Pack

This pack adds the first broader progression layer beyond the individual player rig and current ship.

## Included
- fleet registry scaffolding
- ship progression scaffolding
- assignment and patrol scaffolding
- fleet task state
- meta-loop state for ownership and expansion
- starter fleet, ship progression, and meta definition files

## Goal
Expand the gameplay loop from:
personal survival / salvage / mining

into:
owning ships -> upgrading ships -> assigning roles -> building a fleet -> expanding operational reach
